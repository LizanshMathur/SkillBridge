About SkillBridge

SkillBridge helps professionals and learners connect their career journey with skills development. The platform guides you through a simple flow: Login → Registration → Profile Setup → Dashboard.

How It Works

  • Step 1: Login (index.html) – Sign in with email/password or social login.
  • Step 2: Registration (registration.html) – Create an account, then redirected to profile setup.
  • Step 3: Profile Setup (profile-setup.html) – Complete 3 steps: Current Position, Skills, Career Goals.
  • Step 4: Dashboard (dashboard.html) – Central hub for career development tools.

Common Issues & Fixes

  • Login button not working → Ensure type="submit" and validation.js linked.
  • Profile setup skipping steps → Validation enforced, required fields must be filled.
  • Admin login issues → Requires email, password, and access code.

UI/UX Features

  • Fonts: Poppins, Inter, Roboto.
  • Gradient buttons with hover effects.
  • Responsive layouts for mobile, tablet, and desktop.
  • Smooth progress bar animations.
